# PewS-Project

PewS-Project, é um projeto criado com a finalidade de ser o projeto final da matéria de Engenharia de Software; Neste projeto pretendemos desenvolver as principais funcionalidades de um site, onde seja possível preencher fichas de pacientes (crianças), afim de descobrir de forma precoce as comorbidades do paciente.

Estamos utilizando o aquivo chamado de CONTRIBUTING.md para definir a estrutura do repositório e suas diretrizes.

Nesse repositória até o presente momento, criamos as User Stories, e definimos o graú de dificuldade de cada uma delas.
